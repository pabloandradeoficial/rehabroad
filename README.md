# Rehabroad — IA Diagnóstica para Fisioterapeutas

Plataforma web para fisioterapeutas com:

- prontuário eletrônico
- avaliação e evolução clínica
- apoio clínico com IA
- relatórios em PDF
- agenda e financeiro
- área administrativa
- modo estudante
